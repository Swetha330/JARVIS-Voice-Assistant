#  JARVIS – Voice Controlled Virtual Assistant

##  Project Overview

JARVIS is a voice-controlled desktop assistant developed using Python. It can perform various tasks like opening websites, searching on Google/YouTube, taking screenshots, playing music, and telling the current time using voice commands.

---

##  Features

* 🎤 Voice recognition
* 🔊 Text-to-speech response
* 🌐 Open websites (YouTube, Google, Wikipedia)
* 🔍 Google & YouTube search
* 📸 Screenshot capture
* 🎵 Play music from local system
* ⏰ Tell current time

---

##  Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* pyautogui
* webbrowser
* datetime

---

##  Installation

```bash
pip install -r requirements.txt
```

---

##  How to Run

```bash
python jarvis.py
```
